# Changelog

All notable changes to this project will be documented in this file.

## [0.1.0] - 2026-02-04

### Added
- Kiosk UI with donation tiles, receipt flow, and decline handling.
- Masjid admin portal with tab visibility, Ramadan settings, and quote controls.
- Super admin portal mock for onboarding and registration.
- Ramadan Iftaar day selector with multi-day allocation and progress tracking.
- Zakat flow with custom amount entry and configurable quote.

## [0.2.0] - 2026-02-05

### Added
- Zakat al-Fitr flow with per-person amount, people count, and quote controls.
- Special Appeal tab with progress tracking, presets, and other-amount entry.
- Live reporting table in admin with filters and export (CSV/Excel/PDF).
- Admin configuration panel with per-tab settings and visibility control.
- Ramadan allocation overflow handling and excess summary indicator.
- Admin preview button in kiosk header (demo access on localhost).
