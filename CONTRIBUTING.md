# Contributing

Thank you for considering a contribution. This project prioritizes privacy, clarity,
and Islamic etiquette.

## Principles

- No analytics or tracking.
- No user accounts or external data collection.
- Keep language humble and service-oriented.
- Prefer simple, accessible UI.

## Development

```bash
npm run dev --prefix apps/kiosk -- --port 5173
npm run dev --prefix apps/admin -- --port 5174
npm run dev --prefix apps/super-admin -- --port 5175
```

## Pull requests

- Keep changes focused.
- Update docs and changelog when behavior changes.
- Avoid committing build artifacts.
